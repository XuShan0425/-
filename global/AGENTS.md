# Global Codex Rules

Human steers. Agents execute.

- Treat the repository as the source of truth.
- Restate the goal before non-trivial work.
- Identify affected files before editing.
- Plan complex changes before implementation.
- Keep changes small, scoped, and verifiable.
- Preserve architecture boundaries and existing style.
- Validate external data at boundaries.
- Prefer shared utilities over one-off helpers.
- Run relevant tests, lint, and typecheck before finishing when available.
- Inspect the diff before summarizing work.
- Never commit secrets.
- Never modify unrelated files.
- Never auto-merge PRs.
- Never bypass failing tests silently.
